КОМАНДЫ
winget install --id Cloudflare.cloudflared         - в самом начале нужно скачать сам Cloudflare на сервер (в данном случае на свой ПК)
cloudflared tunnel --url http://localhost:8000     - открыть(создать) тоннель(своего рода прокси) для url = localhost... чтобы он был доступен для всего интернета то новому url

*после выполнения команды в информации в консоли будет новый тоннель с url, который и будет прокси для localhost

cloudflared много что может сделать но тут он открывает доступ для приложение localhost на весь интернет
Можно использовать свой Домен, чтобы открытый тунель был постоянный, а можно использовать Поддомен от самомго cloudflared
*если вы не используете свой собственный домен и вместо этого полагаетесь на поддомен, предоставляемый Cloudflare, то туннель будет временным. Он будет активен только до тех пор, пока не завершится процесс выполнения команды в командной строке. После закрытия или завершения работы командной строки туннель будет автоматически закрыт.
